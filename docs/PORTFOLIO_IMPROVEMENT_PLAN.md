# Heat Chronicle ポートフォリオ改善計画

## 目的

Heat Chronicleを、マツリカのリファラル応募で提示できるポートフォリオとして整える。

新しいAI機能は追加せず、既存プロダクトの完成度、設計力、運用品質、AIを利用した開発プロセスが短時間で伝わる状態を目指す。

- [x] 現在の公開サイトとREADMEを確認する

## スコープ外

今回の応募準備では、以下は実施しない。

- 新しいAI／LLM機能の追加
- バックエンドアーキテクチャの大規模変更
- フロントエンドの全面的なデザイン変更
- 新しいデータ種別やグラフの追加
- 応募に直接必要のないリファクタリング
